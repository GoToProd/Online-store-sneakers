For start project you must:
1. create terminal in VSCode
2. create database in psql
3. cd /path/to/root direction (sneakers)
4. change parametres in .env file
3. run command 'docker-compose build'
4. run command 'docker-compose up'
5. give some time for building database or you can get error 'database is not exist'
